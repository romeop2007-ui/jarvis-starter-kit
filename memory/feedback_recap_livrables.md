À chaque fois qu'un fichier est ajouté dans le dossier `livrables/`, faire un récap immédiatement après.

**Why:** Roméo veut garder une vue claire de ce qui est archivé dans ses livrables sans avoir à fouiller lui-même.

**How to apply:** Après chaque écriture dans `livrables/`, afficher un bloc récapitulatif avec : fichier ajouté, chemin, contenu résumé en une ligne.
