# Livrables

Dossier centralisé pour tous les fichiers produits, documents travaillés et livrables concrets.
Organisation adaptée au profil entrepreneur-étudiant de Roméo.

---

## Conventions

- **Nommage des fichiers** : `YYYY-MM-DD_description-courte.ext` pour les documents datés
- **Fichiers actifs** : dans le dossier racine de la catégorie
- **Archives** : créer un sous-dossier `archives/` dans la catégorie concernée une fois les fichiers obsolètes
- **Imports externes** : les documents bruts à analyser vont dans `context/import/`, pas ici
